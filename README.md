## 🚀 Overview

WitFlix is a front-end web development project built to showcase modern UI/UX principles and API integration. Beyond just a visually appealing layout, WitFlix features **WitBot**—a smart, conversational AI assistant powered by the Google Gemini API. WitBot acts as a personal cinema guide, recommending movies from a curated database based on the user's mood, preferences, and specific prompts.

## ✨ Key Features

* **Sleek UI/UX Design:** Netflix-inspired dark/light modern aesthetic with smooth hover animations and dynamic card layouts.
* **Intelligent AI Chatbot (WitBot):** * Integrated **Gemini 2.5 Flash API**.
  * Context-aware prompt engineering focusing on high customer satisfaction.
  * Recommends movies based on emotional states and specific genres.
* **Custom CSS Architectures:** Utilizing CSS Grid/Flexbox, dynamic background gradients, and text-clamping for perfect aspect ratios.
* **Modular Architecture:** Separated configuration files for API security.

## 🛠️ Built With

* **HTML5** (Semantic structure)
* **CSS3** (Flexbox, Hover Effects, Linear Gradients, Custom Styling)
* **Vanilla JavaScript** (DOM Manipulation, Asynchronous API Fetching)
* **Google Gemini API** (LLM Integration for WitBot)
* **FontAwesome & Google Fonts** (Typography and Icons)

## ⚙️ Installation & Setup (Local Development)

To run this project locally and test the AI chatbot, you need to set up your own Google API Key.

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/yavuzs-cakmak/witflix.git](https://github.com/yavuzs-cakmak/witflix.git)
   cd witflix
Set up the API Key:

Create a file named config.js in the root directory.

Add your Google Gemini API key to this file:

JavaScript
const API_KEY = "YOUR_GEMINI_API_KEY_HERE";
Note: The config.js file is included in .gitignore to prevent API key leaks.

Run the project:
Simply open the index.html file in your preferred web browser, or use an extension like Live Server in VS Code.
